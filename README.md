## About The Project

In this project, we aim to create a simple pipeline for hand-written digits classification using a combination of machine learning and operations tools. The pipeline is built using PyTorch Lightning for the model training, MLflow for tracking and managing the experiment runs, Optuna for hyperparameter tuning, Flask for creating a web API to serve the model, Prefect for workflow management, and Docker for containerization.

The pipeline starts with data preparation, where the dataset is loaded and processed. Next, the model is defined and trained using PyTorch Lightning. During training, Optuna is used to optimize the hyperparameters for better performance. The trained model is then saved and deployed as a web API using Flask, allowing the user to write any digits on the API, and the API returns the classification result using the best model.

Finally, the entire pipeline is containerized using Docker, making it easy to deploy and run on different environments. The pipeline is also managed using Prefect, allowing for complete control over the workflow and easy monitoring of the progress. With the use of these tools, we can achieve an automated and scalable solution for digits classification, providing accurate results in real-time with a simple user interface.

### Installation & Usage

1. Clone the repo
   ```sh
   git clone https://github.com/AymSa/MLOps-Digit-Classification.git
   ```
2. Install requirements.txt
   ```sh
   pip install -r requirements.txt
   ```
3. Train & Save model
    ```sh
    prefect deployment apply main-deployment.yaml
    prefect agent start -q 'train'
    prefect orion start
    ```
4. Transition best model to production
   ```sh
   cd instance
   mlflow server --backend-store-uri sqlite:///mlflow.sqlite
   ```

5. Run Web application (dev for the moment)
   ```sh
   flask --app flaskr run
   ```
